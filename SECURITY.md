# Security Policy

## Supported Scope

현재 저장소는 연구 및 실험 목적의 ROS 2 기반 차량 제어 코드입니다. 다음 범위의 제보를 받습니다.

- 원격 제어, 웹 제어, 토픽 입력을 통한 위험한 동작 유발
- 인증 없이 노출되는 제어 인터페이스
- 민감한 시스템 정보 노출
- 기본 설정만으로 발생하는 안전상 취약점

## Reporting a Vulnerability

공개 이슈로 취약점을 올리지 마십시오. 재현 절차, 영향 범위, 사용한 설정, 하드웨어 조건을 포함해 관리자에게 비공개로 제보하십시오.

연락 채널이 정해지지 않았다면 우선 저장소 관리자에게 직접 연락하고, 공개 저장소에는 수정 완료 후 공개 가능한 범위만 공유하십시오.

## Response Expectations

- 제보 접수 확인
- 영향도 평가
- 재현 가능 여부 확인
- 수정 또는 완화책 정리

응답 속도는 유지보수 가능 시간과 하드웨어 재현 환경에 따라 달라질 수 있습니다.
